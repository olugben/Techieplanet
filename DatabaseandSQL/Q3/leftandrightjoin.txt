A LEFT JOIN returns all rows from the left table, even if there are no matches in the right table.
If no match is found, columns from the right table return NULL.



SELECT games.yr, games.city, city.country
FROM games
LEFT JOIN city
ON games.city = city.name;



RIGHT JOIN


A RIGHT JOIN returns all rows from the right table, even if there are no matches in the left table.
If no match is found, columns from the left table return NULL.
RIGHT JOIN

SELECT games.yr, games.city, city.country
FROM games
RIGHT JOIN city
ON games.city = city.name;
